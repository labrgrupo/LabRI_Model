To ensure the optimal functionality of the LabRI application, please verify that the latest versions of both R and RStudio are installed and properly configured on your system. If these prerequisites are already installed and up to date, you can proceed directly to the installation of LabRI. Otherwise, follow the steps below to set them up:

Steps to Verify or Install:

Verify or Install R:

Check if R is installed on your system by opening a terminal or command prompt and typing R --version. If it is not installed or the version is outdated, visit the official R website (https://cran.r-project.org/) and download the latest version compatible with your operating system. Follow the installation instructions provided on the website.

Verify or Install RStudio:

Check if RStudio is installed and up to date. If it is not installed or the version is outdated, download the latest version from the official RStudio site (https://posit.co/download/rstudio-desktop/).

With R and RStudio verified or installed, you are now ready to proceed with the LabRI application installation.
